# Phase 3: 마이 페이지 탭 (My Page Tab)

## 상태: 대기

## 화면 구성

### 마이 페이지 메인
| 영역 | 기능 | 구현 방식 |
|------|------|----------|
| 프로필 | 닉네임/설정/로그인 수단 | 하드코딩 목업 |
| 복용시간 / 설문 시간 | 알림 시간 표시 | 하드코딩 목업 |
| 나의 약 정보 | 현재 약 종류/용량 리스트 | 하드코딩 목업 |
| 개인정보처리방침 | 법적 고지 | 하드코딩 목업 |
| 버전 | 앱 버전 표시 | 하드코딩 목업 |
| 안내 문구 | 면책 조항 | 하드코딩 목업 |
| 로그아웃 | 로그아웃 버튼 | 하드코딩 목업 |

### 약 정보 수정 (실구현)
| 필드 | 타입 | 필수 |
|------|------|------|
| 약 이름 | 텍스트 입력 | * |
| 용량 | 텍스트 입력 | * |
| 1일 복용 횟수 | 숫자 (1~10) | * |
| 복용 기간 | 텍스트 입력 | 선택 |
| 메모 | 텍스트 입력 | 선택 |
